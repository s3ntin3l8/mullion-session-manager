# CLI logo attribution

The icons in this directory identify third-party CLI tools that this project can
launch as sessions (Claude Code, Codex, opencode, Gemini CLI, Antigravity, Pi Coding
Agent). They are used solely to identify those tools in the session launcher UI and
are not modified beyond format conversion, except `pi-coding-agent-light.svg` (see
below).

`pi-coding-agent-light.svg` is a recolored derivative of the same source
(`pi-coding-agent.svg`, fill changed from white to `#171717`) — dashboard-icons only
publishes one (white-on-transparent) variant, which has poor contrast against this
app's light theme, so a dark-ink twin was made locally for the theme swap, the same
way opencode's native light/dark pair is used elsewhere in this directory.
