# CLI logo attribution

The icons in this directory identify third-party CLI tools that this project can
launch as sessions (Claude Code, Codex, opencode, Gemini CLI, Antigravity). They are
used solely to identify those tools in the session launcher UI and are not modified
beyond format conversion.
